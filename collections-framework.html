Java Collection Framework

In Java, the Collection Framework is a set of interfaces and classes that provide a unified architecture for representing and manipulating collections of objects. Collections in Java are used to store, retrieve, and manipulate groups of objects. The Collection Framework simplifies the process of working with collections by providing common interfaces and classes for various types of collections.

Key Interfaces in the Java Collection Framework

  • Collection Interface:
    • The root interface for all collections.
    • Represents a group of objects known as elements.
    • Common methods include add(), remove(), contains(), size(), etc.
  • List Interface:
    • Extends the Collection interface.
    • Represents an ordered collection that allows duplicate elements.
    • Common implementations include ArrayList, LinkedList, and Vector.
  • Set Interface:
    • Extends the Collection interface.
    • Represents an unordered collection that does not allow duplicate elements.
    • Common implementations include HashSet, TreeSet, and LinkedHashSet.
  • Queue Interface:
    • Extends the Collection interface.
    • Represents a collection for holding elements prior to processing.
    • Common implementations include LinkedList and PriorityQueue.
  • Map Interface:
    • Represents a collection of key-value pairs.
    • Each key must be unique.
    • Common implementations include HashMap, TreeMap, and LinkedHashMap.

Common Classes Implementing These Interfaces

  • ArrayList: A dynamic array implementation of the List interface.
  • LinkedList: A doubly-linked list implementation of the List interface.
  • HashSet: A hash table implementation of the Set interface.
  • TreeSet: A red-black tree implementation of the Set interface.
  • HashMap: A hash table implementation of the Map interface.
  • TreeMap: A red-black tree implementation of the Map interface.

Example Using the List Interface

import java.util.ArrayList;
import java.util.List;

public class CollectionExample {
  public static void main(String[] args) {
    // Creating a List
    List<String> fruits = new ArrayList<>();

    // Adding elements to the List
    fruits.add("Apple");
    fruits.add("Banana");
    fruits.add("Orange");

    // Iterating through the List
    System.out.println("List of Fruits:");
    for (String fruit : fruits) {
      System.out.println(fruit);
    }
  }
}

This is just a basic introduction to the Java Collection Framework. Depending on your needs, you may choose different interfaces and implementations for your specific use case.
